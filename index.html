個人簡介

我是 Calon，目前在高雄的藝術中心擔任展場服務人員。
在五專日文科快畢業時，意識到自己不想以日文為主要工作技能，卻又不知道該往哪去，決定先累積工作經驗，畢業後先在雜貨店當門市人員，之後在網頁前端工程師的哥哥建議之下，接觸了網頁前端，也在嘗試一段時間後有了興趣，並報名了進修部二技資管系學習相關知識，畢業後投了 2、3 月的履歷都沒有下文，這時想加強比較薄弱的後端知識，而在搜尋相關資料時，看到了 WeHelp Bootcamp 的資訊。

  • 從 2020 年 7 月開始，跟著六角學院的課程學習並獨自完成課程最終專案：

如果參與這個計畫，會怎麼安排學習時間？

在去年底看到 WeHelp 的資訊時，雖然已過了報名時間，但也讓我有了準備全職投入時所需的生活費以及學費的時間，工作將會只做到 5 月底，在 7 月課程開始前，會先在網路上學習 MySQL 以及觀看彭彭老師在 Youtube 上的資料結構與演算法直播影片，並且參考歷屆學長姐的專案成果發表會，讓自己的最終專案有一些想法，而在第一階段，因為有學過 HTML、CSS、JavaScript 基礎知識，所以會將學習重心放在網路、網站、後端的知識上面。


在整個課程中，我會在早上注意力比較集中時，先從當週作業和較困難的內容開始著手，自己有接觸過、比較容易理解的內容，則會在下午學習，並且在固定每天下午 4 點半左右出門散步 1 小時，讓腦袋放鬆消化當天所學到的知識。此外，我會將學習內容使用 Anki 工具整理成問題，晚上睡前會使用此工具，以自問自答的方式，複習學習到的知識。

想加入坂和企業有限公司，除了公司福利、薪水好以外，該公司重視人才教育，除了提供給員工成長的課程，也有創立 BoWa 學院培養軟體工程師人才。

請描述一件讓你產生明顯負面情緒的事情，你如何處理該情緒？

我以前是一個跟不熟的人講話就會很緊張、焦慮的人，只要跟別人講話，我就會開始很緊張，並且在之後擔心自己有沒有講錯話，這個性在以前一直困擾著我，尤其五專時最嚴重，因為在課程中會有許多需要與不熟的、不是自己班上的同學練習外文對話，後來因緣際會之下，接觸到了正念冥想，練習察覺自己當下的感受，告訴自己這些擔憂都是自己的想法，然後將注意力拉回到對方講的內容，練習了一段時間之後，當我在跟不熟的人對話並開始焦慮時，都會透過此方法讓自己平復下來，到現在則是能以輕鬆的心情跟陌生人聊天。

關於這份申請網頁，請分享一個你開發時的技術心得。

我會先觀察排版有沒有重複的地方，會將這些重複的排版樣式統一放到一個 class 名稱，在之後相同排版的地方只需要套用該 class 名稱，不需要再額外命名 class 名稱以及撰寫同樣的樣式，例如這份申請表網頁的每個小標題，除了文字、底線顏色之外的樣式都是一樣的，所以我會將小標題共用的樣式寫在 .subTitle 裡面，在每個小標題的標籤套用 .subTitle，這樣我只需要額外撰寫不同顏色的部份即可。
